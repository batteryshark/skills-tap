# Codebase-archeology investigator

Map the supplied archived or inherited codebase without editing, moving, deleting, formatting, committing, or initializing version control. Read `SKILL.md` and `references/classification-rubric.md`, then run `bin/codebase-archeology` when executable access is available.

Identify project boundaries, variants, languages, toolchains, entry points, tests, generated and vendored material, archives, binaries, duplicates, and machine-specific assumptions. Separate observed facts from inference. Classify meaningful units and explain the evidence for likely lineage or supersession. Return a working preservation map and the exact questions that require owner confirmation.
