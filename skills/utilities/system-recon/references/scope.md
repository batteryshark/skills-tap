# Inventory scope

Basic inventory may include OS/version, architecture, uptime, CPU/memory totals, filesystem capacity, and installed command versions. Optional profiles may add interface state, routes, listening sockets owned by the current user, storage topology, or development runtimes.

Exclude credential stores, environment-variable values, shell history, browser data, document contents, private keys, tokens, and unrelated processes' command lines. Redact stable machine identifiers before external sharing.
